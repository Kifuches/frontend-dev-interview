# **Часть 13. System Design (самая модная тема 2026)**

Практически все Senior интервью содержат такой этап.

Например:

Спроектируйте большой интернет-магазин на Angular.

Или:

Нужно сделать CRM на 300 экранов.

Что ожидают услышать:

- структуру проекта;

- управление состоянием;

- разделение ответственности;

- lazy loading;

- кэширование;

- работу с API;

- обработку ошибок;

- авторизацию;

- feature flags;

- переиспользуемые UI-компоненты;

- мониторинг производительности.

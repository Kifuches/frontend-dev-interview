# Название материала

Короткое вступление: что это за тема и почему ее спрашивают на собеседовании.

## Что нужно знать

- Первый ключевой тезис.
- Второй ключевой тезис.
- Практический нюанс из реального проекта.

## Пример

```ts
// Код, если он помогает объяснить тему.
```

## Как отвечать на интервью

Сформулируй ответ так, чтобы было видно понимание причин, trade-offs и практических последствий.
